# sudoku

A simple Java App that solves the sudoku puzzle using artifical intelligence algorithms.
<ul>
	<li>Backtracking</li>
	<li>Forward checking</li>
	<li>MRV (Minimum Remaining Value)</li>
	<li>LCV (Least Constraining Value)</li>
	<li>Minimum Conflict</li>
</ul>
